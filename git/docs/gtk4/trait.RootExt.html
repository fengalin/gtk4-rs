fn set_focus<P: IsA<Widget>>(&self, focus: Option<&P>)[src]

If focus is not the current focus widget, and is focusable, sets it as the focus widget for the root. If focus is None, unsets the focus widget for the root.

To set the focus to a particular widget in the root, it is usually more convenient to use WidgetExt::grab_focus instead of this function.
