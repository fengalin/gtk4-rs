pub fn to_args(&self) -> Option<Bytes>[src]

Creates a new glib::Bytes args from the current state of the given self. Any uniforms of the shader that have not been explicitly set on the self are zero-initialized.

The given ShaderArgsBuilder is reset once this function returns; you cannot call this function multiple times on the same self instance.

This function is intended primarily for bindings. C code should use ShaderArgsBuilder::free_to_args.

Returns

the newly allocated builder, free with gsk_shader_args_builder_free
